Logger:
-------
* logger is a tool that used to log messages to the console.

* logger provides logging functionality for a application it can be used to display error,warning and debug messages.

log-levels:
-----------
1.error
2.warn
3.info
4.verbose
5.debug
6.silly


1.error:- Represents actual error in the code that need attention these messages indicate issues that could affect
          the functionality of the application.

2.warn:- indicate potential issues (or) situation that might lead to problem but don't nessearily cause an error.
         Developers should pay attention to warning messeage.

3.info:- genaral information messages that indicate y=the normal flow of application. these messages are
           useful for traking the execution of the code.
