1.API (Generic Term):

* Definition: Stands for Application Programming Interface.

* Description: An API is a broader term that encompasses various types of interfaces for communication 
  between software components. It can use different protocols, data formats, and architectures.


2.REST API:


* API that conforms to the design principles of the REST, or representational state transfer architectural style.

* Description: A type of API that adheres to the principles of REST, which is an architectural style for 
  designing networked applications. REST APIs use standard HTTP methods (GET, POST, PUT, DELETE) for communication
  and typically operate over the web.
